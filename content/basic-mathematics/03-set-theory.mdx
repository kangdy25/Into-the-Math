# 집합론 (Set Theory)

**집합론**은 수학에서 집합의 개념을 연구하는 분야입니다. 집합은 서로 다른 객체들의 모임을 의미합니다. 집합론은 수학의 기초를 이루는 중요한 이론으로, 수학의 거의 모든 분야에 응용됩니다.

## 집합의 정의

집합은 **원소** 또는 **요소**라고 불리는 객체들의 집합입니다. 집합은 대개 중괄호 `{}`를 사용하여 표현됩니다.

예를 들어, \( A = \{1, 2, 3, 4\} \)는 원소 1, 2, 3, 4로 이루어진 집합 \( A \)를 나타냅니다.

## 집합의 종류

집합에는 여러 종류가 있습니다. 몇 가지 주요 집합의 종류를 살펴보겠습니다:

### 1. 유한 집합 (Finite Set)

유한 집합은 원소의 개수가 **한정된** 집합입니다. 예를 들어:

- \( A = \{1, 2, 3\} \)
- \( B = \{a, b, c, d\} \)

### 2. 무한 집합 (Infinite Set)

무한 집합은 원소의 개수가 **무한한** 집합입니다. 예를 들어:

- 자연수 집합: \( \mathbb{N} = \{1, 2, 3, 4, 5, \dots\} \)
- 실수 집합: \( \mathbb{R} \)

### 3. 공집합 (Empty Set)

공집합은 **원소가 하나도 없는 집합**입니다. 공집합은 \( \emptyset \) 또는 \( \{\} \)로 표현됩니다.

## 집합 연산

집합에는 여러 가지 연산이 있습니다. 주요 집합 연산은 다음과 같습니다:

### 1. 합집합 (Union)

두 집합의 합집합은 두 집합의 원소를 모두 포함하는 집합입니다. 기호로는 \( \cup \)를 사용합니다.

예: \( A = \{1, 2, 3\}, B = \{3, 4, 5\} \)

\( A \cup B = \{1, 2, 3, 4, 5\} \)

### 2. 교집합 (Intersection)

두 집합의 교집합은 두 집합에 모두 포함된 원소들의 집합입니다. 기호로는 \( \cap \)을 사용합니다.

예: \( A = \{1, 2, 3\}, B = \{3, 4, 5\} \)

\( A \cap B = \{3\} \)

### 3. 차집합 (Difference)

두 집합의 차집합은 한 집합에는 포함되지만 다른 집합에는 포함되지 않은 원소들의 집합입니다. 기호로는 \( - \)를 사용합니다.

예: \( A = \{1, 2, 3\}, B = \{3, 4, 5\} \)

\( A - B = \{1, 2\} \)

### 4. 여집합 (Complement)

여집합은 전체 집합에서 특정 집합을 제외한 나머지 원소들의 집합입니다.

## 결론

집합론은 수학에서 매우 중요한 분야로, 여러 수학적 개념들이 집합론을 기반으로 형성됩니다. 집합론을 이해하면 수학의 다른 많은 분야들을 이해하는 데 도움이 됩니다.
